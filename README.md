# 📊 Análise de Tendências de Vídeos

## 🚀 Visão Geral

Este projeto tem como objetivo analisar tendências de vídeos com base em dados de popularidade, engajamento e categorias, utilizando visualizações interativas no Tableau.

A análise busca responder perguntas como:

* Quais tipos de vídeos estão em alta?
* Quais categorias geram mais visualizações?
* Existe padrão de crescimento ao longo do tempo?
* Como o engajamento varia entre diferentes tipos de conteúdo?

---

## 🛠️ Ferramentas Utilizadas

* **Tableau** → Visualização e dashboard interativo
* **Excel / CSV** → Fonte de dados
* **SQL (opcional)** → Tratamento de dados

## 📊 Dataset

O dataset contém informações sobre vídeos, incluindo:

* Título do vídeo
* Categoria
* Data de publicação
* Número de visualizações
* Curtidas (likes)
* Comentários
* Métricas de engajamento

---

## 📈 Análises Realizadas

O dashboard apresenta:

### 📅 Tendência ao longo do tempo

* Crescimento de visualizações
* Identificação de picos de popularidade

### 🎬 Categorias mais populares

* Comparação entre tipos de conteúdo
* Identificação de nichos dominantes

### 👍 Engajamento

* Relação entre visualizações, likes e comentários
* Conteúdos com maior interação

### 🔥 Top vídeos

* Vídeos com maior desempenho
* Ranking por visualizações ou engajamento

---

## 📊 Dashboard

O arquivo `.twbx` contém um dashboard interativo com:

* Filtros dinâmicos (data, categoria, etc.)
* Gráficos de tendência
* Rankings de vídeos
* Indicadores de desempenho
